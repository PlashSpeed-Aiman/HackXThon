# Audio Assets

## Required File
Place a Japanese instrumental music file here named: `ambient-japanese.mp3`

## Recommended Specifications
- **Format:** MP3
- **Length:** 2-5 minutes (for seamless looping)
- **File size:** 2-5 MB
- **Bitrate:** 128-192 kbps
- **Style:** Koto, shakuhachi, zen garden ambience, or similar calm Japanese instrumental

## Free Music Sources (Royalty-Free)

## Installation
1. Download your chosen MP3 file
2. Rename it to `ambient-japanese.mp3`
3. Place it in this directory
4. Rebuild the application

The app expects the file at: `Assets/Audio/ambient-japanese.mp3`
